# `InputEvent`를 확장한 MouseInputEvent 객체

* `type` String - 이벤트의 타입으로 `mouseDown`, `mouseUp`, `mouseEnter`, `mouseLeave`, `contextMenu`, `mouseWheel` 이나 `mouseMove`가 될 수 있다.
* `x` Integer
* `y` Integer
* `button` String (optional) - 눌린 버튼으로 `left`, `middle`, `right`가 될 수 있다.
* `globalX` Integer (optional)
* `globalY` Integer (optional)
* `movementX` Integer (optional)
* `movementY` Integer (optional)
* `clickCount` Integer (optional)
