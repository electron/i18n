# 快速入門

Electron 讓你可以使用 JavaScript 呼叫豐富的原生 (作業系統) APIs 來建立桌面應用程式。 你可以將它視為一個著重於桌面應用程式的 Node.js ，而非網站伺服器。

快速入門文檔已分為兩個文檔:

* 建立一個簡單的Electron app, 請查看 [編寫首個Electron App](./first-app.md)
* 要查看處理程序體系，請查看[主處理程序和渲染處理程序](./application-architecture.md#main-and-renderer-processes)。

了解更多有關Electron，請查閱[官方指南](../)。
