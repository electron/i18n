# Objeto de Referencia

* `url` String - Referencia HTTP de la URL.
* `policy` String - Puede ser: `default`, `unsafe-url`, `no-referrer-when-downgrade`, `no-referrer`, `origin`, `strict-origin-when-cross-origin`, `same-origin` o `strict-origin`.
