* ` issuerCert ` प्रमाणपत्र - जारीकर्ता का प्रमाणपत्र (यदि स्व-हस्ताक्षरित नहीं है)
* ` विषय ` [ CertificatePrincipal ](certificate-principal.md) - विषय मुख्याध्यापक
