# شروع سریع

الکترون شما را قادر می سازد با جاوا اسکریپ خالص و فراهم کردن API های غنی سیستم عامل اصلی، اپلیکیشن دستکتاب بسازید. شما باید این را نوعی از Node.js runtime بدانید که روی ساخت اپلیکیشن دستکتاپ تمرکز کرده به جز بخش وب سرورها.

The old "Quick Start" document that used to live here has been split up into two documents:

* To check out the process architecture, see [Main and Renderer Processes](./application-architecture.md#main-and-renderer-processes).
