# Início Rápido

Electron permite criar aplicações desktop com JavaScript puro, fornecendo em tempo de execução uma API rica e nativa do sistema operacional. Você pode vê-lo como uma variação da runtime do Node.js com foco em aplicações de desktop em vez de servidores web.

A antiga página "Início Rápido" que costumava ficar aqui foi dividida em dois documentos:

* Para ver como é feito um app simples no Electron, confira [Escrevendo Seu Primeiro Aplicativo com Electron][first-app]
* Para ver como é a arquitetura de processos, confira [Processos Principal e de Renderização][processes].

Para aprender mais sobre o Electron, confira os [guias oficiais][readme].
