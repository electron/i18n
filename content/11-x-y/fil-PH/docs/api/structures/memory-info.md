# Mga bagay ng MemoryInfo

* `workingSetSize` Integer - Ang halaga ng memorya na kasalukuyang nakakabit sa aktuwal na pisikal na RAM.
* `peakWorkingSetSize` Integer - Ang pinakamataas na halaga ng memorya na hindi pa nai-pin sa aktwal na pisikal RAM.
* `privateBytes` Integer (optional) _Windows_ - The amount of memory not shared by other processes, such as JS heap or HTML content.

Tandaan na ang lahat ng mga estatistika ay iniulat sa Kilobytes.
