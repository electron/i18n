# Pierwsze Kroki

Electron umożliwia ci tworzenie aplikacji komputerowych przy użyciu czystego JavaScript przez zapewnienie środowiska wykonawczego z bogatymi natywnymi interfejsami API (systemu operacyjnego). Możesz to sobie wyobrazić jako odmiana środowiska wykonawczego Node.js, która jest skupiona na programach komputerowych zamiast na serwerach sieci web.

Stary dokument "Szybki start", który kiedyś tu był, został podzielony na dwa dokumenty:

* Aby sprawdzić, jak prosto jest Electron zbudowany, zobacz [Pisanie Swojej Pierwszej Aplikacji Używając Electron](./first-app.md)
* Aby sprawdzić, architekturę procesu, zobacz [Proces główny i renderer](./application-architecture.md#main-and-renderer-processes).

To learn more about Electron, check out the [official guides](../).
