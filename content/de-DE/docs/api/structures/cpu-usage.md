# CPUUsage-Objekt

* `percentCPUUsage` Number - Prozentzahl der CPU-Leistung, die seit dem letzten Aufruf von getCPUUsage genutzt wurde. Der erste Aufruf gibt 0 zurück.
* `idleWakeupsPerSecond` Number - Die durchschnittliche Anzahl pro Sekunde, wie oft die CPU aus der Inaktivität aufwacht, gemessen seit dem letzten Aufruf von getCPUUsage. Der erste Aufruf gibt 0 zurück.
