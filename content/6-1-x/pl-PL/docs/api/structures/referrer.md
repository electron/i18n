# Obiekt Referrer

* `url` String - HTTP Referrer URL.
* `policy` String - Może być `default`, `unsafe-url`, `no-referrer-when-downgrade`, `no-referrer`, `origin`, `strict-origin-when-cross-origin`, `same-origin` or `strict-origin`. Zobacz specyfikację [Referrer-Policy](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Referrer-Policy) aby uzyskać więcej informacji na temat znaczenia tych wartości.
