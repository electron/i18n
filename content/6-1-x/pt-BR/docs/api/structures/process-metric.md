# Processamento de Objeto

* `pid` Integer - Processo id of proccesso.
* `type` String - Process type. One of the following values:
  * `Browser`
  * `Tab`
  * `Utilidade`
  * `Zygote`
  * `Ajuda ao Sandbox`
  * `GPU`
  * `Pepper Plugin`
  * `Pepper Plugin Broker`
  * `Desconhecido`
* `CPU` [CPUUsage](cpu-usage.md) - uso da CPU.
