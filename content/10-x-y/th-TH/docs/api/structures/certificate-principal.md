* `commonName` String - 通称.
* `organizationUnits` String[] - ชื่อหน่วยขององค์กร.
* `organizationUnits` String[] - ชื่อหน่วยขององค์กร.
* `locality` String - ท้องถิ่น.
* `state` String - รัฐหรือจังหวัด.
* `country` String - ประเทศหรือภูมิภาค.
